# sails-hook-sanseed
Hook for [sails](http://sailsjs.org/)

## About
Hook to add items to the base depending on different localities

## Settings
Install

```
npm install sails-hook-sanseed
```

In `config/seed.js`

```js
module.exports.seed = {
  test: {
    user: [
      {
        //Some data
        username: "juan",
        name: "Paco",
        last: "Pedro",
        password: "de la mar"
      },
      {
        //Some data
      }
    ]
  },
  production: {
    user: [
      {
        //Some data
        username: "test",
        name: "Test",
        last: "Test",
        password: "iamademo"
      },
      {
        //Some data
      }
    ]
  },
  serverX: {
    //Some data
  }
}
```

In `config/models.js`

```js
module.exports.models = {
  //If you want to drop all the time
  migrate: 'drop'
};
```

## Use
Test at `/seed/production` or `/seed/test/user`
